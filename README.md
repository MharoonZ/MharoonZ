## 🚀 Featured Projects

### 🏥 Agentic Clinical Trial PDF-to-FHIR Publisher

> **Problem:** Medical researchers manually extracted clinical trial data from hundreds of PDFs — slow, error-prone, expensive.
>
> **Solution:** Full agentic AI pipeline. PDFs in → Google Gemini extracts structured trial metadata with schema-controlled prompting → HL7 FHIR/EBM-compliant JSON out → Published automatically via FEvIR REST API. Human review layer built in for clinical safety.
>
> **Impact:** ~70% reduction in manual extraction time. 20+ clinical trial document types handled.

`Python` `Google Gemini` `HL7 FHIR` `FEvIR REST API` `Streamlit` `JSON Schema`

[![Live Demo](https://img.shields.io/badge/Live_Demo-7C3AED?style=for-the-badge&logo=vercel&logoColor=white)](https://fevir-ui.vercel.app)
[![Clinical Trials](https://img.shields.io/badge/CT.gov_System-14B8A6?style=for-the-badge)](https://ct-gov.ms-strategies.com)

---

### 💊 AI Clinical Decision Support System — Heart Failure

> **Problem:** Heart failure treatment requires checking dozens of guideline rules per patient from unstructured doctor notes.
>
> **Solution:** GenAI system that reads clinical notes → extracts 10+ parameters (LVEF, NYHA class, meds, labs) using hybrid NLP + AWS Comprehend Medical → maps to ACC/AHA guidelines → GPT-4 generates safe personalized recommendations. Deterministic logic owns clinical rules. AI owns the language.
>
> **Impact:** Consistent accuracy across 5+ case types. Zero hallucinated clinical recommendations.

`Python` `GPT-4` `AWS Comprehend Medical` `NLP` `Streamlit` `JSON`

[![Live Demo](https://img.shields.io/badge/Live_Demo-7C3AED?style=for-the-badge&logo=streamlit&logoColor=white)](https://aws-medical.streamlit.app)

---

### 📈 AI Quantitative Research & Backtesting Platform

> **Problem:** Quant research client needed a unified platform for market data, backtesting, strategy scoring, and AI-assisted reporting.
>
> **Solution:** Modular FastAPI backend with separate layers for data ingestion, analysis requests, task coordination, and report generation. AI workflows for strategy evaluation and performance comparison. Delivered in 2-week sprint.
>
> **Impact:** Full production delivery for international client. Architected for MT5 integration.

`Python` `FastAPI` `Streamlit` `Pandas` `REST APIs` `AI Workflows`

[![Live Demo](https://img.shields.io/badge/Live_Demo-7C3AED?style=for-the-badge&logo=vercel&logoColor=white)](https://fact-weld.vercel.app)

### ⚡ Cloud-Native LLM Inference System

> **Problem:** Running open-source LLMs in production at low latency without burning cloud budget.
>
> **Solution:** OpenAI-compatible `/v1/chat/completions` endpoint on Modal GPU infrastructure with vLLM. Persistent model cache, async scheduling, GPU memory tuning. Any OpenAI SDK connects without code changes.
>
> **Impact:** Sub-200ms inference latency. Runs Gemma, Llama, Mistral variants.

`Python` `vLLM` `CUDA` `Modal GPU` `Hugging Face` `OpenAI-Compatible API`

[![Code](https://img.shields.io/badge/View_Code-181717?style=for-the-badge&logo=github&logoColor=white)](https://github.com/MharoonZ/gemma4-modal-vllm-inference)
